# Студенческий проект на тему игрового ПО в жанре RTS

## Концепция игры
Игроку предстоит взять на себя роль главы поселения людей, обороняющихся от нападений титанов. Побеждая титанов поселение получает ресурсы, необходимые для своего развития.

## Основные механики:
- Враги волнами нападают на поселение с некоторой периодичностью;
- За убийство титанов даются ресурсы на развитие поселения;
- Под развитием поселения подразумевается расширение, укрепление стен, а также строительство зданий для увеличения количества людей;
- В армию входит только определенный процент всех людей;
- За те же ресурсы с титанов возможно улучшение армии в виде увеличения этого процента или бонусов (увеличение здоровья или наносимого урона);
- В самом поселении происходит разделение на кланы, игрок должен выбрать один из них. Существует военный клан (дает бонусы к армии) и демократический клан (увеличивает количество людей, появляющихся после постройки дома).
